Design notes — 02 · Terminal

Direction: a developer-console feel. Each surface is a terminal window with a title bar, the primary action reads as a command (❯ share --current-page), links are a menu, and the options page is a config file of key = value rows with square ON/OFF toggles. A faint CRT scanline (tinted from the text color) sits over everything.

Typography: monospace throughout — the closest system stack to the extension's real vendored Inconsolata. A second monospace stack is offered as an alternate.

Palettes: Phosphor green (the current UI's #41ff00 lineage), Amber (vintage CRT), and Ice cyan. Dark is the native mode; each palette also ships a readable "paper terminal" light variant.
