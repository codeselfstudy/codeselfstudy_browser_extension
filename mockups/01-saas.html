Design notes — 01 · Modern SaaS

Direction: a clean, friendly product UI — rounded cards, a soft accent-colored primary action, generous spacing, and subtle elevation. The popup reads as a focused action sheet; the options page as a settings panel.

Typography: a neutral system sans for body with the same family for headings at heavier weights — invisible, legible, gets out of the way. A rounded system stack is offered as an alternate for a softer feel.

Palettes: Indigo (default) is confident and modern; Emerald leans calm/organic; Slate is the safe neutral. Each defines all seven tokens for light and dark, so dark mode is a pure variable swap.
